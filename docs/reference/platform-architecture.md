## ğŸ¯ Executive Summary

**Erynoa** ist eine Service-Marktplatz-Plattform, die auf einer **Orchestrierungs-basierten Architektur** aufbaut. Der Kern besteht aus:

| Aspekt             | Technologie | Zweck                             |
| ------------------ | ----------- | --------------------------------- |
| **Backend**        | Rust (Axum) | Hochperformante API               |
| **Frontend**       | SvelteKit   | Moderne, reaktive UI              |
| **Auth**           | ZITADEL     | Enterprise Identity Management    |
| **Orchestrierung** | Temporal.io | Workflow-Management & Reliability |
| **Datenbank**      | PostgreSQL  | Persistente Datenhaltung          |
| **Cache**          | DragonflyDB | High-Performance Caching          |
